Inverse Trig & Triangle-from-Ratio

The asymmetry arcsin and arctan answer negatives with negative angles. arccos answers negatives in QII — its range [0, π] contains no negative angles. This single fact decides most evaluation questions.

Evaluation recipe

  1. Find the reference angle from the magnitude (special-angle table).
  2. Input positive → done, it's the QI angle.
  3. Input negative → arcsin/arctan: negate the reference angle. arccos: π − reference.

Compositions

sin(sin⁻¹ x) = xalways, for x in [−1, 1] — no trap in this direction
sin⁻¹(sin θ) = θonly if θ ∈ [−π/2, π/2]
cos⁻¹(cos θ) = θonly if θ ∈ [0, π]
tan⁻¹(tan θ) = θonly if θ ∈ (−π/2, π/2); otherwise shift by multiples of π

Out of range? Compute the inner value, then find the in-range angle sharing it. Worked folds: sin⁻¹(sin 2π/3) = π/3, cos⁻¹(cos 5π/4) = 3π/4, tan⁻¹(tan 5π/6) = −π/6.

Triangle-from-ratio recipe

  1. The given ratio names two sides — write them on a right triangle (sin: opp/hyp · cos: adj/hyp · tan: opp/adj).
  2. Pythagoras the third side.
  3. Read off the requested function — the triangle gives the magnitude.
  4. If θ is placed outside QI, set the sign from the quadrant (ASTC below).

Mixed compositions (cos of arcsin, etc.)

cos(sin⁻¹ x) = √(1 − x²)blueprint: opp x, hyp 1
sin(cos⁻¹ x) = √(1 − x²)blueprint: adj x, hyp 1
tan(sin⁻¹ x) = x/√(1 − x²)blueprint: opp x, hyp 1
sec(tan⁻¹ x) = √(1 + x²)blueprint: opp x, adj 1

Don't memorize these — re-derive in ten seconds: name the inner angle θ, draw its blueprint with hypotenuse (or adjacent) 1, Pythagoras, read off.
