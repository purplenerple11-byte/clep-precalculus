Graphing Sinusoids

Companion to Lesson 0003. Print it; keep it beside you while drilling. Grounded in Paul's Online Math Notes — Graphs of Trig Functions.

The standard form

y = A · sin(B(x − C)) + D   (or cos)

SymbolNameEffect on the graph
Aamplitude factoramplitude = |A|; if A < 0 the curve flips vertically
Bangular frequencyperiod = 2π/B; bigger B ⇒ shorter period
Cphase shiftshifts right if C > 0, left if C < 0 — after B is factored out
Dvertical shiftmidline is y = D

Read-off formulas

amplitude|A| = (max − min) / 2
midliney = D = (max + min) / 2
period2π / B
B from a known periodB = 2π / period
maximumD + |A|
minimumD − |A|
The two traps Period is 2π/B, never B.   Phase shift needs B factored out first: in sin(2x − π) factor to sin(2(x − π/2)), so the shift is π/2, not π.

Five-key-points method (to sketch one cycle)

  1. Draw the midline y = D and the max/min lines at D ± |A|.
  2. Start of cycle: x = C. Mark the four following points a quarter-period apart: C, C + T/4, C + T/2, C + 3T/4, C + T where T = 2π/B.
  3. For sin: the pattern across those five points is midline → max → midline → min → midline.
  4. For cos: max → midline → min → midline → max.
  5. If A < 0, flip the max/min roles.

sin ⇄ cos

cos x = sin(x + π/2)cosine is sine shifted left by a quarter period
sin x = cos(x − π/2)sine is cosine shifted right by a quarter period

Sanity checks

The curve never leaves [D − |A|, D + |A|]an answer with points outside that band is wrong
Exactly one full cycle per 2π/Bif your sketch shows a different count, B is off
Amplitude is a distanceit is never negative — read |A|
