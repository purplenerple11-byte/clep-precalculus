Counting Trig Solutions

Companion to Lesson 0007. Print it. Grounded in Paul's Online Math Notes — Solving Trig Equations.

The substitution habit

  1. Name the inside: u = whatever sits inside the function (2x, 3x, 2x − π…).
  2. Stretch the window: x ∈ [0, 2π) and u = Bx ⇒ u ∈ [0, 2πB) — B full circles.
  3. Solve circle by circle: first-circle solutions, then +2π each per extra circle.
  4. Map back: divide every u by B (undo the full inner expression if it was Bx − φ).

Say it aloud Crossings twice per circle. Peaks once. Impossible never. Tangent: once per period, no exceptions.

Spacing checks

tan(Bx) = csolutions evenly spaced π/B apart — an uneven list means a missing root
sin(Bx) = c, cos(Bx) = csolutions come in pairs per cycle of length 2π/B; each pair repeats shifted by 2π/B
